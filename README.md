# nvim

Installation:

    - git clone https://github.com/skyfant/nvim/ $HOME/.config/
    
    - init.lua: comment out all required('') files except of 'plugins'.
    
    - init.lua: uncomment all commented required('') files.



Uninstall:

    - rm -rf ~/.config/nvim

    - rm -rf ~/.local/share/nvim
